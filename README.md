# Lagsphp
LagsPHP, inspired from _TOF Lags refactoring KATA (https://github.com/ToF-/LagsJava)

## Install

```bash
composer install
```

## Execute

```bash
php lags.php
```

## Unit test

```bash
vendor/bin/phpunit
```

## With code coverage

```bash
vendor/bin/phpunit --coverage-html ./report
```
